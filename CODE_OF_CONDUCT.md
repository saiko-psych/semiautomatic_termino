# Code of Conduct

This project follows the **Contributor Covenant, version 2.1**.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

The Code of Conduct applies within all project spaces (issues, pull
requests, discussions, commit messages) and also applies when an
individual is officially representing the project in public spaces.

## Reporting

Reports of violations can be sent to the project maintainer at the
contact email listed in `README.md` (search for "Contact" or the
maintainer's email at the top of the file).

Reports will be reviewed and investigated promptly and fairly.

## Enforcement

The project maintainer follows the Contributor Covenant Enforcement
Guidelines:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
